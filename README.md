Clean-Portfolio

🚀 Features
🎨 Clean and professional UI/UX design
📱 Fully responsive (Mobile, Tablet, Desktop
⚡ Smooth scrolling 
📄 Resume download option
🔗 Social media links (LinkedIn, GitHub)

HTML5
